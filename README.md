This is a repository for my personal website showcasing my web development projects, blog posts, and contact information. The website is built using HTML, CSS, and Bootstrap, and is hosted using GitHub Pages.

Installation

To install this website locally, you can clone this repository by running the following command in your terminal:

git clone https://github.com/your-username/your-repo-name.git

Usage

You can view the website by visiting the following URL:

https://alexguynichols.github.io/

The website has a navbar that links to the different pages, including the home page, about me page, contact page, and projects page. You can also view my blog posts on the blog page.

Contributing

If you have any suggestions or find any issues with the website, feel free to open an issue or submit a pull request. I welcome any contributions!
